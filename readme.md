# Using FASTMCP

FASTMCP is a tool for Minecraft modding development.

## Setup Instructions

1. Download FASTMCP from the official source
2. Extract the files to your desired location
3. Run the setup script
4. Wait for decompilation to complete

## Key Features

- Fast decompilation process
- Built-in development environment
- Automatic workspace setup
- Compatible with modern Minecraft versions

## Notes

- Make sure to have Java installed
- Keep original files backed up
- Follow the documentation for best practices

For more information, check the official documentation.
